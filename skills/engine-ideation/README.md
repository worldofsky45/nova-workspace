# Engine Ideation - Automated Opportunity Discovery

**Purpose:** Weekly generation of self-improving predictive engine ideas

## What This Does

**Every Sunday 9 PM (automated via HEARTBEAT):**
1. Reviews what's working across current engines
2. Generates 2-3 new engine ideas
3. Scores them against proven criteria
4. Recommends top idea to Sky
5. Logs to `~/engine-ideas/weekly-YYYY-MM-DD.md`

## Why This Matters

**Instead of:** You asking "what else can we build?"

**Now:** Nova proactively surfaces opportunities every week based on:
- What's working in existing engines
- Market trends
- Data availability
- Fast validation potential
- Low capital requirements

## The Pattern (Reusable)

Every successful engine has:
1. **Data Collection** - APIs, scraping, feeds
2. **Analysis** - Pattern detection, predictions
3. **Execution** - Auto or alert for manual
4. **Tracking** - JSONL logs, scorecards
5. **Learning** - Weekly/monthly calibration
6. **Alerts** - Telegram, dashboard

**Proven with crypto-intel. Now replicate across niches.**

## How New Ideas Get Generated

### Criteria (Non-Negotiable)
- ✅ Clear feedback loop (measure right/wrong objectively)
- ✅ Frequent iterations (daily/weekly, not yearly)
- ✅ Quantifiable outcomes ($ made, time saved, accuracy %)
- ✅ Self-improvement possible (patterns emerge from data)
- ✅ Automation potential (runs without constant human input)

### Red Flags (Avoid)
- ❌ Outcomes take >3 months to measure
- ❌ Success depends on uncontrollable factors
- ❌ No clear performance tracking
- ❌ Requires large upfront capital
- ❌ Market dominated by better-funded competitors

## Weekly Ideation Process

1. **Review active engines:**
   - What's working? (crypto showing 65% win rate)
   - What's not? (content not driving newsletter subs yet)
   - What patterns emerged? (fear zones work for BTC)

2. **Look for adjacent opportunities:**
   - Similar data sources (can we use X for Y?)
   - Similar prediction types (price movements → engagement?)
   - Similar feedback loops (daily tracking → same pattern elsewhere?)

3. **Generate 2-3 ideas:**
   - Use engine template
   - Score against criteria
   - Estimate ROI + risk

4. **Recommend top 1:**
   - Surface to Sky via Telegram
   - Include one-line pitch + ROI model
   - Link to full write-up
